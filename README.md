## Project Setup

Install NodeJS if not installed: https://nodejs.org/en](https://nodejs.org/en)

### Install dependencies
```
npm install
```

## Running application

Create a file called `.env` in the backend directory with the following contents:

```
MONGO_URI=xxx
JWT_SECRET=xxx
PORT=xxx
ADMIN_PASS=xxx
```

### Run in development mode
```
npm run dev
```

### Run normally
```
npm start
```
